🧠 Prediction Methodology

The core logic uses a weighted heuristic algorithm that favors recurring interest. A single visit is treated as exploration, while multiple visits within a short time window significantly boost the "Purchase Probability" score.

Privacy is central: No browsing data ever leaves your local machine. All processing and analysis happen within the Flask environment running on 127.0.0.1.
