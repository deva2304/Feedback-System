CCLF Model Pipeline

Correlation-Constrained Prediction

Powered by SVD decomposition of behavioral correlation matrices — not simple heuristics.

🔌
1. Extension Captures
Every product page load is tracked. Revisits increment the view count — a strong buying signal. Add-to-cart events are captured automatically.
🔬
2. Feature Extraction
8 behavioral features are extracted: views, duration, price, recency, velocity, cart status, category affinity, and relative interest.
📊
3. Correlation Matrix
Pearson correlation matrix is built across all products, then decomposed via SVD into latent factors that capture behavioral patterns.
🧠
4. CCLF Prediction
Correlation-constrained dot product + sigmoid calibration produces a realistic purchase probability. More views & cart = higher score.
